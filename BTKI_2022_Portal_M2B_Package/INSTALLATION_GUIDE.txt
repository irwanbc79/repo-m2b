## ðŸš€ Quick Start (30 Menit)

### Step 1: Database Setup (5 menit)
```bash
# Copy migration file
cp database/create_btki_tables_migration.php \
   YOUR_LARAVEL_PROJECT/database/migrations/2024_01_01_000001_create_btki_tables.php

# Run migration
cd YOUR_LARAVEL_PROJECT
php artisan migrate
```

### Step 2: Import Data (10 menit)
```bash
# Install Python dependencies
pip install pandas openpyxl mysql-connector-python

# Run importer
cd python/
python btki_importer.py \
    --excel "PATH_TO/E-BTKI 2022 v1 - 1 April 2022.xlsx" \
    --db-host localhost \
    --db-user root \
    --db-password YOUR_PASSWORD \
    --db-name portal_m2b
```

### Step 3: Laravel Integration (10 menit)
```bash
# Copy Laravel files
cp laravel/models/HsCode.php YOUR_LARAVEL_PROJECT/app/Models/
cp laravel/controllers/HsCodeApiController.php YOUR_LARAVEL_PROJECT/app/Http/Controllers/Api/
cp laravel/livewire/HsCodeSearch.php YOUR_LARAVEL_PROJECT/app/Http/Livewire/
cp laravel/views/hs-code-search.blade.php YOUR_LARAVEL_PROJECT/resources/views/livewire/

# Install Livewire (jika belum)
cd YOUR_LARAVEL_PROJECT
composer require livewire/livewire

# Clear cache
php artisan config:clear
php artisan cache:clear
php artisan view:clear
```

### Step 4: Configure Routes (5 menit)

Add to `routes/web.php`:
```php
use App\Http\Livewire\HsCodeSearch;

Route::get('/hs-code', HsCodeSearch::class)->name('hs-code.search');
```

Add to `routes/api.php`:
```php
use App\Http\Controllers\Api\HsCodeApiController;

Route::prefix('v1')->group(function() {
    Route::get('/hs-codes/search', [HsCodeApiController::class, 'search']);
    Route::get('/hs-codes/{hsCode}', [HsCodeApiController::class, 'show']);
    Route::post('/hs-codes/validate', [HsCodeApiController::class, 'validate']);
    Route::get('/hs-codes/chapters', [HsCodeApiController::class, 'chapters']);
    Route::get('/hs-codes/chapter/{chapterNumber}', [HsCodeApiController::class, 'byChapter']);
    Route::get('/hs-codes/hierarchy/{hsCode}', [HsCodeApiController::class, 'hierarchy']);
    Route::get('/hs-codes/general-rules', [HsCodeApiController::class, 'generalRules']);
    Route::get('/hs-codes/popular', [HsCodeApiController::class, 'popular']);
    Route::get('/hs-codes/autocomplete', [HsCodeApiController::class, 'autocomplete']);
});
```

### Step 5: Test!
```bash

curl http://localhost/api/v1/hs-codes/search?q=ikan

# Access UI
http://localhost/hs-code
```
